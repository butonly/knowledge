# DevOps

DevOps（Development-Operations）是一种重视`“软件开发人员（Dev）”`和`“IT运维技术人员（Ops）”`之间沟通合作的文化、运动或惯例。透过自动化`“软件交付”`和`“架构变更”`的流程，来使得`构建、测试、发布`软件能够更加地`快捷、频繁和可靠`。
